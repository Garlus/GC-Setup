# Garlicus Computer - Setup


## Features

- **Flatpak Apps** — Install curated apps from Flathub across categories

- **GNOME Shell Extensions** — Install popular extensions directly from extensions.gnome.org
- **System Packages** — Get copyable terminal commands for native packages with automatic detection of your package manager (apt/dnf/pacman/zypper)
- **Font Rendering** — One-click sharp font rendering with Apply/Revert buttons
